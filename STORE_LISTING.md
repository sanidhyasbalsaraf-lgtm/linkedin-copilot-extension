## Full description

```
Outreach Copilot reads the job description and hiring-contact details off a
LinkedIn job posting you're viewing, then drafts a connection-request note,
InMail, or job-reply cover note — written in your own voice, not a generic
template.

HOW IT WORKS
• Open a LinkedIn job posting. Click the on-page "OC" button (bottom-right
  corner) or the toolbar icon.
• Capture the job description and hiring-manager info with one click.
• Pick a message type and click Generate — the draft is written using your
  own background, tone, and real writing samples (set once in Settings).
• Insert it straight into LinkedIn's own message box, or copy it yourself.

BRING YOUR OWN OPENAI KEY
Drafts are generated by calling OpenAI directly from your browser, using an
API key you provide (get one at platform.openai.com/api-keys). Usage is
billed to your own OpenAI account at OpenAI's rates. This extension is free,
open source, and never bundles, proxies, or shares a key with anyone.

PRIVACY
Your API key and writing samples are stored only in your browser's local
storage — never synced, never sent anywhere except directly to OpenAI when
you click Generate. No analytics, no tracking, nothing sold. Full privacy
policy: see the link on this listing page.

Independent tool — not affiliated with, endorsed by, or sponsored by
LinkedIn Corporation or OpenAI. Source code is public on GitHub.
```

## Per-permission justifications

Paste the relevant one into each permission's justification box:

- **storage** — Saves your OpenAI API key, writing-style settings, and
  appearance preference locally on your device, so you don't have to
  re-enter them every time.
- **activeTab** — Reads the job description / hiring-contact info from the
  LinkedIn tab you're currently viewing when you open the popup or click
  Re-scan/Capture.
- **scripting** — Injects the content script into the current LinkedIn tab
  on demand, so Re-scan and Insert work reliably even on tabs that were
  already open before the extension was installed or updated.
- **host permission: `https://www.linkedin.com/*`** — Required to read job
  postings, hiring-team/profile info, and to insert the drafted message into
  LinkedIn's own compose box, across job pages, profile pages, and the
  messaging inbox.
- **host permission: `https://api.openai.com/*`** — Required to send the
  drafting request directly to OpenAI using the user's own API key when they
  click Generate.

## Data-use disclosures (Privacy Practices tab)

Check **only** these two categories as collected:
- **Website content** — job/hiring-contact text read from LinkedIn pages.
- **Authentication information** — the user's own OpenAI API key.

Leave all other categories (personally identifiable information, health,
financial/payment, personal communications, location, web history, user
activity) **unchecked** — none of this data is read, stored, or transmitted.

For both checked categories, the required affirmations are all true:
- Used only for **the extension's core functionality** (drafting messages),
  not for any unrelated purpose.
- **Not sold** to third parties.
- **Not used or transferred for creditworthiness or lending purposes.**
